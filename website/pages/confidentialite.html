Politique de Confidentialité

La présente politique de confidentialité décrit comment Sijill Project ("nous", "notre" ou "l'application") collecte, utilise et protège vos informations personnelles lorsque vous utilisez notre application mobile et notre site web.

1. Informations collectées

Nous collectons les types d'informations suivants :

  • Informations de compte : nom, adresse email, mot de passe (crypté)
  • Données d'utilisation : cours consultés, progression d'écoute, préférences
  • Informations de paiement : traitées de manière sécurisée par Stripe (nous ne stockons pas vos données bancaires)
  • Données techniques : type d'appareil, système d'exploitation, adresse IP

2. Utilisation des informations

Vos informations sont utilisées pour :

  • Créer et gérer votre compte utilisateur
  • Fournir l'accès aux contenus audio et aux cours
  • Traiter vos paiements et gérer vos abonnements
  • Personnaliser votre expérience d'apprentissage
  • Envoyer des communications relatives à votre compte (emails de bienvenue, réinitialisation de mot de passe)
  • Améliorer nos services et développer de nouvelles fonctionnalités

3. Partage des informations

Nous ne vendons jamais vos données personnelles. Vos informations peuvent être partagées avec :

  • Stripe : pour le traitement sécurisé des paiements
  • SendPulse : pour l'envoi d'emails transactionnels
  • Cloudflare : pour l'hébergement sécurisé des contenus

Ces partenaires sont tenus de protéger vos données conformément aux réglementations en vigueur.

4. Sécurité des données

Nous mettons en œuvre des mesures de sécurité appropriées :

  • Chiffrement des mots de passe (bcrypt)
  • Connexions HTTPS sécurisées
  • Authentification par tokens JWT
  • Accès restreint aux données personnelles

5. Conservation des données

Vos données sont conservées tant que votre compte est actif. Vous pouvez demander la suppression de votre compte et de vos données à tout moment en nous contactant.

6. Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Accès : obtenir une copie de vos données personnelles
  • Rectification : corriger vos informations inexactes
  • Suppression : demander l'effacement de vos données
  • Portabilité : recevoir vos données dans un format structuré
  • Opposition : vous opposer au traitement de vos données

Pour exercer ces droits, contactez-nous à contact@sijillproject.com

7. Cookies

Notre site web utilise des cookies essentiels pour :

  • Maintenir votre session de connexion
  • Mémoriser vos préférences

Nous n'utilisons pas de cookies publicitaires ou de tracking tiers.

8. Application mobile

Notre application mobile peut demander les permissions suivantes :

  • Internet : pour accéder aux contenus en streaming
  • Audio en arrière-plan : pour continuer l'écoute lorsque l'app est en arrière-plan

9. Modifications

Nous pouvons mettre à jour cette politique de confidentialité. Les modifications seront publiées sur cette page avec une nouvelle date de mise à jour.

10. Contact

Pour toute question concernant cette politique de confidentialité :

  • Email : contact@sijillproject.com
  • Site web : sijillproject.com
